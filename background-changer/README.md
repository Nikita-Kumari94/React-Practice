# Background Changer (React Practice Project)

A simple React practice project that allows users to change the background color of the page using buttons.  
Built to understand **React state management and event handling**.

---

## Tech Stack

- React
- JavaScript (ES6)
- CSS
- Vite / Create React App

---

## Learning Outcomes

This project helped me practice:
- React functional components
- `useState` hook
- Handling click events
- Dynamic styling in React
- Basic React project structure

---

Made for practice, fun, and getting more comfortable with React.
